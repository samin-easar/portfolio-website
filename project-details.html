TeleHeal 24/7 Health
Care App

TeleHeal 24/7 is an Android application that connects patients with doctors for online consultations. Patients can register, book appointments, and chat with doctors securely. The app uses Firebase for user authentication, real-time database, and storage to ensure a smooth and secure experience.

Technologies Used

Java JavaFX Firebase Android Studio

Key Features

  • User Authentication
  • Real-time Consultation Sync
  • Online Doctor Assisting
  • Secure Data Storage Core
